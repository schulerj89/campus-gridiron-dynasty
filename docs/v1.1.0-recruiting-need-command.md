# v1.1.0 Recruiting Need Command

## Expert Decision

The CFB game expert recommended Recruiting Need Command as the first minor feature after v1.0.0 because roster scarcity already drives the sim, but the recruiting page did not make every positional gap visible at once.

## Why

Dynasty recruiting should be a needs-aware decision loop. The player should see where the roster is short, how much of that shortage is already covered by the board, which positions have scholarship offers out, and where committed players are already solving the problem. This keeps the existing recruiting model intact while making the interface more strategic.

## What Changed

- Added command cards for all positions on the Recruiting page.
- Each card shows current non-senior roster count, target count, need, active board count, active user offers, and user pledges.
- Clicking a card filters the recruiting database to that position.
- Auto-recruit behavior remains unchanged.

## Acceptance

- Recruiting page shows all position needs.
- Need cards show current/target, active board, offers, and pledges.
- Clicking a position need card filters the recruiting database.
- Smoke artifacts include `artifacts/screenshots/recruiting-needs-desktop.png`.
