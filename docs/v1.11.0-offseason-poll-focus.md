# v1.11.0 Minor Release: Offseason Poll Focus

## Decision

The offseason Overview dashboard now hides the Current Poll panel, matching the existing offseason hiding behavior for Dynasty Command and Latest National Awards.

## Why

The offseason dashboard should focus on the active stage: departures, recruiting, signing day, or development. Poll context remains available in the Rankings tab, so keeping it on the offseason dashboard added noise.

## Verification

- Added smoke coverage that verifies the offseason departures dashboard hides command, awards, and poll panels.
- Refreshed offseason dashboard screenshots.
- Ran release, build, unit, and smoke validation for the version.
