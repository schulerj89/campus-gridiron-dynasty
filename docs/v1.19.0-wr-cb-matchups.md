# v1.19.0 WR Speed And CB Matchups

## Decision

Treat this as a minor simulation release because player generation and game outcomes both changed.

## Why

The expert audit found two connected issues: wide receivers aged into low-speed rooms because recruits were capped too tightly, and individual corner matchups were not part of target or play-by-play decisions. Fixing only one side would either keep future WR rooms slow or make CBs unable to keep up with speed recruits.

## Implementation

- WR and CB generation now includes athletic archetypes.
- WR/CB recruit speed can exceed the recruit entry cap up to 93, while recruit overall and non-speed attributes remain capped by trait band.
- Aggregate target, receiving-yard, and receiving-touchdown splits now use a receiver-versus-coverage multiplier.
- Play-by-play target selection, completion chance, and pass-play yardage use the same matchup edge.
