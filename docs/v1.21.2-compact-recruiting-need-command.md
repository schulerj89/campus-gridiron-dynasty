# v1.21.2 Compact Recruiting Need Command

## Release Type

Patch release. This is a Recruiting page UI polish change with no recruiting logic, storage, or save-shape changes.

## Code-Review Expert Decision

The code-review expert flagged the Need Command area as a major contributor to Recruiting page length. The old implementation rendered every position as a tall card even though the data was mostly compact numeric coverage.

## Changes

- Replaced tall Need Command cards with a denser position matrix.
- Preserved roster target, need, board, offer, pledge, and coverage meter data for every position.
- Preserved click-to-filter behavior for each position.
- Added a compact legend so the shortened row labels stay readable.
