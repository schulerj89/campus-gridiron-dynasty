# v1.21.7 Focused Roster Depth

## Release Type

Patch release. This is a UI simplification for a long roster/depth chart page.

## Code-Review Expert Decision

The code-review/design expert found that Roster Room stacked the 85-player roster list and every depth card on the same page. The page kept all features, but the default experience was visually heavy.

## Why

Roster browsing and depth-chart management are different tasks. A focused view switch keeps both features available while reducing page length and making screenshots easier to inspect.

## Changes

- Added a `Roster List` / `Depth Chart` switch inside Roster Room.
- Compactified depth cards to top three visible players per position.
- Kept hidden reserves reachable by allowing the visible cutoff player to move down when more players exist at that position.
- Added smoke coverage for the hidden-reserve demotion path.
