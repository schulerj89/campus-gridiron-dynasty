# v1.21.6 Effective Roster Order

## Release Type

Patch release. This is a logic/UI consistency fix for roster ordering.

## Code-Review Expert Decision

The code-review/design expert found that depth charts used effective overall while the roster list still sorted and labeled players by base overall. Hot and cold streaks could make the same position group appear differently between the roster and depth chart.

## Why

Roster browsing should match the rating value that already drives depth-chart order. Showing `Eff` for streak-adjusted rows keeps the temporary form effect visible without changing the player profile's base overall.

## Changes

- Sorted roster rows by `effectiveOverall`, then potential, then base overall.
- Labeled adjusted roster rows as `Eff` and preserved base overall in title text.
- Updated depth-chart tests to compare effective overall and verify each slot reports its full position count.
