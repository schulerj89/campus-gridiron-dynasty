# v1.20.0 Offseason Focus Flow

## Decision

Treat the offseason dashboard as a sequence of focused review checkpoints instead of a mixed recap page. The user should see one primary job at a time: departures, recruiting, signing day, development, then program review.

## Implementation

- Added `departuresReviewed` and `programReviewComplete` markers to offseason reports.
- Kept the existing simulation rules, but added a no-sim departures review click before offseason recruiting starts.
- Added a preseason program review click between development results and regular-season kickoff.
- Updated the top advance button to show stage-specific labels and an immediate disabled loading state.
- Replaced the limited development list with a full roster table that shows overall deltas, potential, incoming freshman/walk-on status, and attribute movement.

## Validation

- Added desktop smoke screenshots for championship recap and each focused offseason stage.
- Updated simulator regression coverage for the new departures and program-review checkpoints.
- Ran build, unit tests, release check, and smoke coverage for the visible version bump.
