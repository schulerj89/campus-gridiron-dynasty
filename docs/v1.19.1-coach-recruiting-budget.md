# v1.19.1 Coach Recruiting Budget

## Decision

Treat this as a patch release because it fixes accounting for an existing coach-point action.

## Why

Coach recruiting ratings already feed the season recruiting budget, but spending a coach point on recruiting did not refresh the active budget. That made the upgrade look disconnected until another lifecycle event rebuilt recruiting points.

## Implementation

- `spendCoachPoint` now calls the existing recruiting budget reconciliation path when the upgraded skill is `recruiting`.
- Non-recruiting coach-point upgrades keep their previous behavior.
- Sunk recruiting points remain spent while `pointsRemaining + pointsSpent` reconciles to the updated season budget.
