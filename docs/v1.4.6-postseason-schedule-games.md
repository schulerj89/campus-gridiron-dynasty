# v1.4.6 Postseason Schedule Games

## Decision

The Schedule tab now displays both regular-season schedule games and playoff games from `state.playoff.games`.

## Why

Matchup preview already found pending playoff games, but the Schedule table only read `state.schedule`. In postseason that made playoff games visible in the preview while the table had no playoff row or box-score access.

## Scope

- Added a Schedule display selector that merges regular-season and playoff games.
- Prioritized user-team games before current-week fillers so the row limit does not hide completed user games.
- Added smoke coverage that forces a playoff, opens Schedule, verifies a bowl row, and captures a postseason Schedule screenshot.

## Validation

- `npm run smoke`
