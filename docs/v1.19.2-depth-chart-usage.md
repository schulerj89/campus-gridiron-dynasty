# v1.19.2 Depth Chart Usage

## Decision

Treat this as a patch release because it fixes an existing roster control that was being persisted but not used by simulation selection.

## Why

The depth chart let users move players, but game usage still selected by rating order. That meant a manually promoted player could stay unused, making the depth chart feel cosmetic.

## Implementation

- Game selection helpers now accept full teams and apply saved depth-chart order within each position.
- Aggregate stat allocation and play-by-play context both use the same depth-chart-aware helpers.
- Rating order remains the fallback when no manual order exists.
