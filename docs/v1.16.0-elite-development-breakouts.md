# v1.16.0 Elite Development Breakouts

Date: 2026-06-18
Release type: Minor

## Decision

Update offseason player development so elite and high-potential returners can make meaningful breakout jumps, including movement into the 90+ overall range.

## Why Minor

This changes long-term dynasty progression and roster-building outcomes, so it is a gameplay feature/balance release rather than a patch.

## Scope

- Added trait-specific development profiles for base growth, breakout odds, focus gain caps, and max visible overall gain.
- Added breakout focus-attribute bursts for players who hit the development roll.
- Added attribute-driven overall catch-up so large real attribute gains can produce visible OVR jumps.
- Kept player development capped by potential and kept the no-regression invariant.
- Added a real offseason advancement regression test for elite 85 OVR / 96 POT returners reaching 90+.

## Integrity Notes

Incoming freshmen remain excluded from preseason development. Overall catch-up is only applied after attribute gains and remains capped by player potential.
