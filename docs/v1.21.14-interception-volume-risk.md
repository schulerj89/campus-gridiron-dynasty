# v1.21.14 Interception Volume Risk

## Release Type

Patch release. This is a game simulation logic fix.

## Code-Review/Stats Expert Decision

The code-review/stats expert found that interceptions only used passing-vs-coverage edge. Air Raid and run-heavy teams with identical ratings could carry the same total pick risk despite very different pass volumes.

## Why

Turnover totals should reflect opportunity. High-volume passing should create more total interception exposure while keeping per-attempt rates bounded by quarterback quality and coverage strength.

## Changes

- Added an estimated pass-attempt input to interception generation.
- Kept the risk curve capped so higher volume does not create runaway interception rates.
- Added aggregate strategy coverage comparing Air Raid and run-heavy pass attempts, interceptions, and interception rate.
