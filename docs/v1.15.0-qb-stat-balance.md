# v1.15.0 QB Stat Balance

Date: 2026-06-18
Release type: Minor

## Decision

Rework QB completion percentage and passing-touchdown distribution. Completion rate is now centered on QB accuracy, QB awareness, receiving support, pass protection, opposing coverage, and strategy. Passing touchdowns now receive a stronger strategy-aware share for Air Raid and spread systems.

## Why Minor

This changes season stat pacing and strategy outcomes, which are core gameplay behavior rather than a narrow bug fix.

## Scope

- Added a named completion-rate helper in `src/sim/game.ts`.
- Added a named passing-touchdown-share helper in `src/sim/game.ts`.
- Lowered the default completion-rate center so average and weak passing rooms are no longer pushed into the high clamp.
- Preserved total score and total offensive touchdowns while changing pass-vs-run touchdown split.
- Added tests for weak passing attributes and Air Raid touchdown share.

## Integrity Notes

Completion percentage remains derived from completions and attempts. The scoring plan still determines total touchdowns; strategy only affects how offensive touchdowns are assigned.
