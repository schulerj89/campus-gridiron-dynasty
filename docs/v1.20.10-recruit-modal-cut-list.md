# v1.20.10 Recruit Modal Cut List

## Expert Finding

The code-review expert found that the regular recruit modal recomputed a top-ten list from all interest scores even after a recruit had narrowed to a smaller Top 5 or Top 3. That made the modal disagree with the recruit's saved `topSchools` state.

## Decision

Treat this as a patch-level UI integrity fix. It aligns display with existing recruit state and does not change recruiting math, commitment odds, storage shape, or weekly progression.

## Fix

- Added `rankedRecruitSchoolInterests` to preserve a recruit's valid `topSchools` cut list.
- Added a sorted-interest fallback for older or malformed saves with missing school IDs.
- Updated `RecruitModal` school-interest rows and user-rank messaging to use the shared helper.
- Added regression coverage for valid cut-list order and invalid cut-list fallback order.
