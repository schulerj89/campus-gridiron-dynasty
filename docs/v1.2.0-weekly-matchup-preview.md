# v1.2.0 Weekly Matchup Preview

## Expert Decision

The CFB game expert recommended Weekly Matchup Preview as the second minor feature because the weekly schedule already existed, but the user had to open rows and infer stakes manually.

## Why

Dynasty mode needs week-to-week context. A pending game should tell the player who is next, where the game is played, whether there are conference or ranked-opponent stakes, and which roster units have the edge. This adds game-week storytelling without changing the simulation engine.

## What Changed

- Added a shared matchup preview helper based on the current schedule, rankings, team power, and unit ratings.
- Added a dashboard preview for the next pending user-team game.
- Added the same preview to the Schedule page above the games table.
- Preserved existing box-score modal behavior.

## Acceptance

- Overview shows the next pending user matchup when available.
- Schedule shows the same matchup context while keeping game rows clickable.
- Preview handles no-pending-game states.
- Unit edge comparisons use existing ratings.
- Smoke artifacts include `artifacts/screenshots/dashboard-next-game-desktop.png` and `artifacts/screenshots/schedule-matchup-preview-desktop.png`.
