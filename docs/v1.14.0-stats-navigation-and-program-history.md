# v1.14.0 Stats Navigation and Program History

Date: 2026-06-18
Release type: Minor

## Decision

Move stat leaderboards out of Awards and into a dedicated Stats tab. Move the program record book and dynasty history into the Program page.

## Why Minor

This changes the app navigation and adds a new user-facing page. It is more than a patch because users now access statistical analysis through a separate tab with broader scope controls.

## Scope

- Added `src/components/StatsView.tsx` for leaderboard filtering and display.
- Added National, every conference, and User Team leaderboard scopes.
- Added derived completion percentage as a selectable stat with minimum-attempt qualification.
- Added `src/components/ProgramHistoryView.tsx` so record book and dynasty history live under Program instead of Awards.
- Updated smoke screenshots for the new Stats page and Program history placement.

## Integrity Notes

Completion percentage remains derived from completions and attempts, not stored. Awards no longer imports or owns leaderboard/history logic.
