# v1.12.0 Minor: Simulation Strategy Playbook

## Decision

Ship the requested simulator changes as a minor release because they add saved-state fields, new game-result payloads, visible strategy controls, and expanded stat contracts.

## What Changed

- Offensive strategies now influence pass/run volume while still considering roster strength and game state.
- Player stats now separate field goals from extra points and track receiver targets.
- Completed games now include scoring play-by-play for simulation diagnostics.
- Program Blueprint has focus presets for balanced, recruiting, development, academics, facilities, retention, and custom plans.

## Why

The previous game engine backfilled stats from a final score without enough visibility into how scoring happened. Separating FG/XP, tracking targets, exposing strategy, and showing play-by-play makes the sim easier to tune and easier for users to understand.

## Verification

- Added game tests for targets, strategy pass/run shifts, score composition, and play-by-play final scores.
- Added dynasty tests for weekly play-by-play/stat output, Blueprint focus presets, and offensive strategy changes.
- Added storage migration coverage for older saves missing new fields.
- Updated smoke coverage and screenshots for strategy, Blueprint focus, box scores, play-by-play, targets, and XP columns.
