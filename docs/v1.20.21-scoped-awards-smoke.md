# v1.20.21 Scoped Awards Smoke

## Expert Finding

The code-review expert found that the forced-award smoke assertion used a global `user-team-award-card` locator. Weekly award panels and season award panels can both render user-team award cards, so a global assertion could pass against the wrong panel.

## Decision

Treat this as a patch-level test hardening change. It does not change product behavior, persisted data, or simulation logic.

## Fix

- Scoped the forced-award assertion to `awards-panel`.
- Added an explicit `Iron Lantern Trophy` expectation inside the Season Awards panel.
- Kept existing screenshots and player modal award coverage intact.
