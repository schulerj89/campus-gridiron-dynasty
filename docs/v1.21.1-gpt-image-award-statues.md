# v1.21.1 GPT-Image Award Statues

## Release Type

Patch release. This fixes the visual quality of the `v1.21.0` award statue assets without changing award logic or candidate-board behavior.

## Decision

The first statue set was too placeholder-like for the award presentation. The replacement set uses file-backed `gpt-image-2` generation through the image generation CLI so the committed assets are actual generated PNGs rather than local fallback drawings.

## Asset Direction

- Polished bronze football trophy statuettes.
- Distinct pose per national season award.
- Neutral pose for the Iron Lantern Trophy.
- Dark product-render background that fits the existing awards card frame.
- No real schools, logos, readable text, or real award likenesses.
