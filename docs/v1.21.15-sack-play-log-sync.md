# v1.21.15 Sack Play Log Sync

## Release Type

Patch release. This is a bug fix with a small simulation refactor.

## Code-Review/Stats Expert Decision

The code-review/stats expert found that defensive sack stats and play-by-play sack events were rolled independently. A defense could receive sack credit that never appeared in the game log, or the log could show sacks missing from the box score.

## Why

Sacks affect awards, leaderboards, offensive-line trust, and play-by-play credibility. The box score and play log need to tell the same story.

## Changes

- Captured credited defensive sackers in each team game profile.
- Passed the opponent's sack queue into offensive play-by-play context.
- Made pass plays consume that queue for sack events instead of rolling independent play-log sacks.
- Added deterministic coverage comparing each defense's sack total to the opponent's play-by-play sack count.
