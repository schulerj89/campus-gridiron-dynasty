# v1.12.11 Championship Recap

## Decision

Classified as a patch release because the simulator already recorded `playoff.championTeamId`; the issue was that the dashboard moved into the offseason report without surfacing that completed championship result.

## Changes

- Added a championship recap panel to the offseason Overview when the Crown Bowl has just completed.
- Shows the champion, final score, runner-up, champion record, completed bracket, and navigation to Playoff Center or Schedule.
- Added a champion banner to the reusable playoff bracket when a champion name is available.
- Added smoke coverage and a dedicated screenshot for the post-final championship recap.

## Validation

- Unit coverage confirms the completed playoff and champion remain available after the final round advances into offseason.
- Smoke coverage checks the championship recap before the departures-focused offseason dashboard flow continues.
