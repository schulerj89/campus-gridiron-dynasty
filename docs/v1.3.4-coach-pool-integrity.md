# v1.3.4 Coach Pool Integrity Decision

## Code-Review Expert Finding

`hireCoach` installed the selected coach and removed that coach from `coachPool`, but the replaced user coach was not returned to the pool. Over a long dynasty, manual hires could silently delete coach records.

## Patch Decision

When the user hires a coach, return the displaced coach to `coachPool` with `hiredBy` cleared.

## Why

- Coach carousel integrity matters over a 20-year dynasty.
- Hiring should exchange staff with the market rather than destroy the outgoing coach.
- The fix keeps the coach pool size stable and preserves future hiring options.

## Validation

- Added a postseason hiring regression proving the hired coach is installed, removed from the pool, and the displaced coach returns to the pool.
- Smoke refreshes postseason/staff screenshots with the visible patch version.
