# v1.21.0 Award Statue Candidate Boards

## Release Type

Minor release. This adds a new visual awards presentation and a new candidate-board experience on the Awards page.

## Decision

Season awards now share a single definition list for award key, display name, and eligibility scope. The same definitions power winner selection, candidate-board generation, and trophy image selection, which reduces the chance that the UI and sim logic drift apart.

Each national season award displays a bronze football statue image in the card. The Iron Lantern Trophy uses a neutral pose, while the position awards use different football action poses.

## Image Generation Note

The built-in image generator was used for the bronze statue art direction. The generated output was not exposed as a recoverable project file in this session, so stable in-repo PNG assets were created under `public/assets/award-statues/` following that generated direction.

## Candidate Boards

Active Week 8 award watch and finalized season awards show the top eight candidates for each award. Boards include rank, player, school, position, year, overall, and a score based on the same player-value model used for awards.

Completed-season history still shows the statue cards, but does not attach a current candidate board because historical player rankings are not stored in season history yet.

## Validation

- Added unit coverage for candidate-board count, rank ordering, QB eligibility, defensive back eligibility, and freshman-only filtering.
- Added smoke coverage for statue image rendering and top-eight candidate board visibility during active season awards.
