# v1.10.0 Minor Release: Signing-Day Recruit Modal

## Decision

All Team Classes rows on signing day are now clickable. Each signed prospect opens a read-only modal with the signee summary, hidden trait, overall, potential, full attributes when available, and final school-interest context.

## Why

Signing day is the payoff for recruiting, so the class table should support inspection instead of acting like a static report. The modal stays read-only because recruiting actions are already resolved at this point.
