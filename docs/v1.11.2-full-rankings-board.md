# v1.11.2 Patch: Full Rankings Board

## Code-Review Finding

The Rankings page rendered `latest.entries`, which contains only Top 25 teams. Poll snapshots already store `allEntries` for every team, so programs outside the Top 25 still lacked a visible ranking.

## Decision

Render `latest.allEntries` with a Top 25 fallback for older saves. Keep moved-in and moved-out panels focused on Top 25 movement.

## Why Patch

This is a presentation logic fix using state that already exists. It does not alter poll generation, saved data, or matchup ranked-label behavior.
