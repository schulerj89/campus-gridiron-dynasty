# v1.9.0 Minor Release: Other-Team Roster Viewing

## Decision

Roster Room now includes a program selector that can inspect any team's roster and depth chart. The selected team summary shows conference, overall, team identity, record, helmet, and roster power. Depth-chart movement remains locked to the user program.

## Why

Dynasty context needs scouting and comparison outside the user roster. Letting the player view other programs supports recruiting decisions, playoff preparation, and roster balance checks without making CPU team depth charts accidentally editable.
