# v1.12.1 Patch Decision - Program Blueprint Auto Build

## Decision

Keep the Program Blueprint `Auto Build` action enabled before kickoff even when the current plan has no remaining points.

## Why This Is a Patch

This fixes an existing control state bug. The button rebuilds the plan, so it should remain available while the Blueprint is editable, including after a focus preset fills every point. It does not add a new gameplay system or change the point-allocation rules.

## Integrity Notes

- The underlying allocator already supports rebuilding a full plan by clearing the current allocation and distributing the season points again.
- The UI now disables `Auto Build` only when the Blueprint is locked.
- Smoke coverage verifies the button remains enabled after selecting a preset that spends every available point.
