# v1.21.13 Scoring Plan XP Composition

## Release Type

Patch release. This is a game simulation bug fix.

## Code-Review/Stats Expert Decision

The code-review/stats expert found that `scoringPlan` accepted the first exact score composition it found. Scores like 24 could become two touchdowns, four field goals, and zero made extra points instead of a normal three-touchdown, one-field-goal line.

## Why

Missed extra points should come from kicker accuracy, not from score-fitting artifacts. Preferring normal made-XP compositions keeps kicker stats and box scores credible.

## Changes

- Added scoring-plan tie-breaks that prefer fewer missed extra points, then more offensive touchdowns, then fewer field goals when score distance is equal.
- Exported `scoringPlan` for focused regression coverage.
- Added tests for common scores: 24, 30, 31, 38, and 45.
