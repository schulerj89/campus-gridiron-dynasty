# v1.20.11 Recruit Interest Scale

## Expert Finding

The code-review expert found that recruit interest values were displayed as percentages even though the underlying recruiting score can reach 150. That could produce visible values above 100%, which made the UI look mathematically wrong.

## Decision

Treat this as a patch-level UI integrity improvement. It changes labels only; recruiting calculations, sorting, interest meters, and save data stay unchanged.

## Fix

- Added `formatRecruitInterest` for visible recruiting interest text.
- Updated the recruiting database, recruit cards, recruit modal, and signed recruit modal to show `score/150`.
- Kept interest meter widths based on `score / 150`.
- Extended smoke coverage to assert `/150` appears in the recruiting database and modal.

- `npm run test`
- `npm run release:check`
- `npm run smoke`
