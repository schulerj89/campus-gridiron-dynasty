# v1.13.0 Stat Attempts and Pancakes

## Decision

Classified as a minor release because it adds new persisted stat fields and changes game simulation output. Completion percentage remains derived from completions and attempts instead of being stored.

## Changes

- Added `passAttempts`, `passCompletions`, and `rushAttempts` to `PlayerStats`.
- Quarterbacks now receive completions and attempts; rushers receive carry counts.
- Box scores, player cards, award notes, and the current leaderboard surface attempts, completions, completion percentage, rushing attempts, and pancakes.
- Reworked pancake distribution from small rounded shares into a direct blocker-weighted count so offensive linemen can record visible pancake stats.
- Positive rushing play-by-play can now cite the blocker whose pancake created the lane.
- Legacy save normalization infers missing attempts and completions from older passing/rushing lines.

## Validation

- Unit coverage checks stat invariants, legacy stat backfills, and pancake play-by-play evidence.
- Smoke screenshots refresh box score, play-by-play, awards, leaderboard, and player profile coverage.
