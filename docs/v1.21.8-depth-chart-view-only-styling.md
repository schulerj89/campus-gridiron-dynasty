# v1.21.8 Depth Chart View-Only Styling

## Release Type

Patch release. This is a UI styling and affordance fix.

## Code-Review Expert Decision

The code-review/design expert found that other-team depth charts rendered disabled up/down controls on every row. The row grid also reserved too little width for two compact icon buttons, which made spacing fragile.

## Why

Disabled controls on view-only CPU rosters looked like unavailable actions instead of read-only scouting information. Removing them makes the mode clearer, and using an auto-sized action column prevents editable rows from crowding.

## Changes

- Rendered depth movement controls only for the user's program.
- Changed depth rows to use an auto-sized action column with a minimum button width.
- Added smoke coverage that switches to another team and asserts movement buttons are absent.
