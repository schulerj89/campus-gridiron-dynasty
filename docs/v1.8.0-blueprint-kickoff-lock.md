# v1.8.0 Minor Release: Program Blueprint Kickoff Lock

## Decision

Program Blueprint points remain editable through Week 1 setup, then lock for the season when Week 1 advances. If the user leaves points unspent, the sim now auto-fills only those remaining points using the same team-needs logic as Auto Build while preserving any manual allocations.

## Why

The prior behavior let a player accidentally carry unused Program Points into the season, which made the annual planning system less clear and could leave recruiting, scouting, and development boosts idle. The new behavior keeps the planning decision lightweight: adjust it before kickoff, or let the staff fill the gaps once games start.
