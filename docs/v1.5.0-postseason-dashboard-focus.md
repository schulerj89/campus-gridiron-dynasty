# v1.5.0 Postseason Dashboard Focus

## Decision

The Overview dashboard now switches into a postseason mode where the playoff bracket is the first and primary panel.

## Why

When the postseason starts, the user needs bracket state and round advancement more than regular-season dashboard summaries. Awards, poll details, and schedule views remain reachable from the main navigation.

## Scope

- Promoted the playoff bracket to the top of Overview during postseason.
- Added an Advance Round button to the bracket panel.
- Hid regular-season command, latest awards, matchup preview, and current-poll panels during postseason.
- Added smoke assertions for the focused postseason dashboard.

## Validation

- `npm run smoke`
