# v1.11.1 Patch: Signing-Day Signee Pagination

## Code-Review Finding

The signing-day class panel rendered only `signees.slice(0, 14)`. Larger classes can reach 22-28 signees, so recruits outside the first page could not be opened in the signed prospect modal.

## Decision

Add pagination to `ClassSigneesPanel` with the existing shared `PaginationControls` component. Reset the signee page when the selected team changes.

## Why Patch

This is a contained UI bug fix for the v1.10.0 signing-day drill-in feature. It does not change simulation data or saved-state shape.
