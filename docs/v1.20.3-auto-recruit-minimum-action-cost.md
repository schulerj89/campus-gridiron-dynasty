# v1.20.3 Auto Recruit Minimum Action Cost

## Expert Finding

The code-review expert found that the manual Auto Recruit button and several automatic advance-week checks used the scholarship cost as the minimum threshold. That stranded 40-point remainders even though the auto-recruit engine could still scout a prospect.

## Decision

Treat this as a patch-level UI and control-flow integrity fix. Auto recruit should be available whenever it can perform at least one valid recruiting action.

## Fix

- Added `MIN_RECRUITING_ACTION_COST` to the recruiting service.
- Updated the auto-recruit loop, manual Auto Recruit button, offseason auto-recruit check, and refunded-points reallocation check to use the shared threshold.
- Added a regression test that verifies auto recruit spends a 40-point remainder on scouting when it cannot afford an offer.
