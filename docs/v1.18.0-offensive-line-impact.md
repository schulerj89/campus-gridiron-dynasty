# v1.18.0 Offensive Line Impact

## Decision

Treat this as a minor simulation release because offensive line ratings now change generated game outcomes directly, not just presentation.

## Why

The expert audit found that line quality already influenced rushing and aggregate sack totals, but the effect was muted and did not reach pass-play generation. That made elite offensive lines feel interchangeable with average lines, especially in box scores and play-by-play.

## Implementation

- Pass protection edge now feeds aggregate passing yards per attempt.
- Pass protection edge now changes play-by-play sack chance.
- Pass protection edge now changes play-by-play completion chance.
- A controlled regression test replays the same matchup with a strong line and weak line, proving the strong line improves rushing, passing efficiency, and sack prevention.
