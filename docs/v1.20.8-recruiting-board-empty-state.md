# v1.20.8 Recruiting Board Empty State

## Expert Finding

The code-review expert found that the Recruiting Board showed `0/35` while rendering the first six matching database prospects as cards. Those prospects were not actually on the board, which made the board state misleading.

## Decision

Treat this as a patch-level UI integrity improvement. It corrects presentation of existing recruiting state without changing recruiting mechanics, storage, or board limits.

## Fix

- Render actual board entries only in the Recruiting Board grid.
- Show an explicit empty board state when the board has no active recruits.
- Added a small reusable `.empty-state` style for compact panel messages.
