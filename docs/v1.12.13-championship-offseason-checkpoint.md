# v1.12.13 Championship Offseason Checkpoint

## Decision

Classified as a patch release because the championship recap and offseason report already existed. This fixes the transition timing so the two views no longer compete on the same dashboard step.

## Changes

- The Crown Bowl final now advances to a postseason championship recap checkpoint.
- The checkpoint preserves the champion, final score, and completed bracket without creating the offseason report yet.
- The next advance opens offseason departures, recruiting bonuses, and the normal offseason dashboard focus.
- The recap includes an `Advance to Offseason` action for a clear next step.

## Validation

- Unit coverage verifies the final round stops on the recap checkpoint and the following advance creates the offseason report.
- Smoke coverage captures the championship recap separately, then advances and captures the focused offseason departures dashboard.
