# v1.12.10 Per-Play Play-By-Play

## Decision

Classified as a patch release because the box score already had play-by-play, but it only recorded scoring, kick, and turnover summaries. This update corrects that feature so the log represents each simulated drive at play granularity.

## Changes

- Added normal rush, pass, sack, punt, touchdown, field goal, extra point, missed kick, and interception rows to completed game play-by-play.
- Added metadata for play number, down, distance, yard line, and yards so the UI can show real per-play context.
- Added punt descriptions with distance, returner or no-return result, and return yards.
- Added interception descriptions with the passer, interceptor, and return yards.
- Kept scoring math and immediate extra-point ordering intact.

## Validation

- Unit coverage now requires full play logs to include normal offensive snaps, down-distance metadata, punts, and interception return detail.
- Smoke screenshots refresh the completed game box score and play-by-play panel.
