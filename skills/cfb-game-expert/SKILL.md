# CFB Game Expert

## Workflow

1. Check current official sources before claiming what recent CFB games include.
2. Convert inspiration into fictional mechanics only: no real schools, real players, real awards, real logos, or licensed marks.
3. Favor mechanics that create dynasty decisions: recruiting scarcity, revealed traits, staff movement, budget tradeoffs, program identity, and historical records.
4. Keep the browser game playable quickly. Prefer readable summaries over console-style menu depth when the feature cannot fit a web UI.

## Mechanics Lens

- Recruiting should include uncertainty, scouting spend, late decisions by top prospects, needs-aware automation, and gem/bust reveals after signing day.
- Program building should use annual or weekly resource points that can be invested in staff, academics, facilities, training, wellness, and recruiting reach.
- Coaching should include pools, coordinator movement, firings, hires, skill boosts, and job pressure driven by expectations.
- Season flow should surface weekly awards, polls, conference races, bowl/playoff qualification, and end-of-year history.
- Player ratings should affect simulation through position-specific attributes, not just one overall number.
